Here's how to convert this template into your own website
  • Open the 'Website Files' folder.
  • Open the 'index' file in any browser. you'll see the website opened.
  • Now open the same file in any text editor.
  • Now press 'ctrl + F' to open the find dialog box.
  • In the find dialog box enter the words 'CHANGE HERE' (all caps) and press enter. it will find every occurence of these words in the document.
  • Now update the content under every occurence of these words.
  • Now repeate the same for 'portfolio' page by opening the 'portfolio' file in the text editor.
  • Now in order to replace the dummy images. open the 'images' folder and delete dummy images and paste your images here in order to be shown on website.
    NOTE ( All the names of the images you add should be same as the dummy images. And all images should be in 'jpg' format )
  • After all this done succesfully. Your website is now ready to host on live server. Upload all the files inside the 'Website Files' folder to your web hosting. Connect the domain name and thats it.
